Konfigurasjon

Konfigurering gjøres vha

  • Attributter

    sette data- attributter på ett input element med klassen tbacinput og vente til autoinitiering av alle elementer med klassen tbacinput kjører i $(document).ready eventen: 
    <input class="tbacinput" data-actype="pro">
  • og/eller

  • Konfig-objekt

    velge ett input element med jQuery, overstyre default-konfig (minimum actype) parametre vha i ett konfig javascript-objekt og så starte manuelt: 
    var konfig = { actype: 'pro' };
    $('#testboks').tbac(konfig)

Eneste som kreves er typen boks settes.

HTML (Attributter)

Minimum

Attributten data-actype satt til en gyldig verdi

<input class="tbacinput" data-actype="pro">

Vanligste

data-actype

  • avd: Avdeling
  • arb: Arbeidstype
  • ans: Ansatt
  • pro: Prosjekt
  • prl: Prosjektlinje
  • akt: Aktivitet
  • art: Lønnsart
  • hav: Hovedavdeling (må implementeres)
  • el1: Element 1 (må implementeres)
  • el1: Element 1 (må implementeres)

data-verdiSetter initiell verdi.


<input 
    class="tbacinput" 
    data-actype="pro" 
    data-verdi="1117"

data-tekstSetter navnet/teksten på valgt post


<input 
    class="tbacinput" 
    data-actype="pro" 
    data-verdi="1117" 
    data-tekst="Taktekking">

data-filterEn WHERE-setning (uten WHERE).

Filtrering av data på server

Eksempel: AvdelingID='1' AND ErAvdelingsGruppe=False


<input 
    class="tbacinput" 
    data-actype="pro" 
    data-filter="Utfort=false" >

data-filtermalEn WHERE-setning med placeholder-tekst som erstattes av verdier fra andre Tbac bokser dynamisk .

Eksempel: AvdelingID='?avd' AND ErAvdelingsGruppe=False


<input 
    class="tbacinput" 
    data-actype="akt" 
    data-filtermal="Oppgavenr='?pro'" 
    data-filterkilde="#prosjektfilterBoks">

data-filterkildeKilde for verdi som flettes inn i data-filtermal

CSS-selector til ett select, input, tbacinput eller hvilket som helst element med en verdi tilgjengelig via $(element).val() eller $(element).data('verdi').

Verdien til denne boksen erstatter ?abc i data-filtermal og resultatet erstatter statisk filter satt med data-filter attributten og brukes på samme måte

Eksempel: <input class="tbacinput" data-actype="akt" data-filtermal="OppgaveNr='?pro'" data-filterkilde="#prosjekt-velger"> vil filtreres med filteret data-filter="OppgaveNr='1000'" om prosjekt nr 1000 var verdien til en select/input med id="prosjekt-velger" for eksempel.


<input 
    class="tbacinput" 
    data-actype="akt" 
    data-filtermal="Oppgavenr='?pro'" 
    data-filterkilde="#prosjektfilterBoks">

Kode (Javascript)

Minimum

Konfig-objekt med gyldig verdi i actype property

/* Gitt HTML: <input id="prosjektTbac"> */
$('#prosjektTbac').tbac({ actype: 'pro' });

Dette er standardinnstillingsobjektet. Alle kan overstyres ved å sende med ett konfig objekt ved aktivering.
